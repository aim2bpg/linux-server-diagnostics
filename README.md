# Linux Server Diagnostics

Linuxサーバー診断コマンドのビジュアルリファレンス

![Site Image](https://raw.githubusercontent.com/aim2bpg/linux-server-diagnostics/main/images/site_image.png)

## Demo

https://aim2bpg.github.io/linux-server-diagnostics/

## 特徴

- CPU、メモリ、ディスク、ネットワーク、プロセス、Docker等、17カテゴリ・約63コマンド収録
- 各コマンドに目的、オプション詳細、実行例を記載
- ワンクリックでクリップボードにコピー可能

## 必要なパッケージ

一部のコマンドは別途パッケージのインストールが必要です:

- **sysstat** - sar, mpstat, iostat
- **docker** / **docker.io** - docker関連コマンド
- **lsof** - ポート・ファイル使用状況確認
- **htop**, **iotop** - 対話的なプロセス/I/O監視
